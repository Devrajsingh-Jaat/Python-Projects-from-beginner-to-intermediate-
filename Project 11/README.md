# 🧮 Python Calculator

A simple command-line calculator built in Python that supports basic arithmetic operations: addition, subtraction, multiplication, and division. It allows continuous calculations using previous results or restarting a new session anytime.

---

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Continuous calculations using previous results
- Option to restart a new calculation
- ASCII art logo using `art` module
- Simple command-line interface
