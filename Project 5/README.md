# PyPassword Generator

A simple Python program that generates a secure random password based on the number of letters, symbols, and numbers chosen by the user.

## Overview

The **PyPassword Generator** is a beginner-friendly Python project that creates a randomized password.  
Users can customize their password by specifying how many:

- Letters
- Symbols
- Numbers

The program then randomly selects characters from predefined lists and shuffles them to produce a secure password.

## Features

- Custom password generation
- Random mix of letters, numbers, and symbols
- Password characters are shuffled for better security
- Command-line interaction
- Beginner-friendly Python project

## Requirements

- Python 3.x
- Uses only Python’s built-in `random` module

## How the Program Works

1. The program greets the user.
2. The user enters:
   - Number of letters
   - Number of symbols
   - Number of numbers
3. Random characters are selected from predefined lists.
4. All characters are stored in a list.
5. The list is shuffled to randomize the order.
6. The characters are combined to form the final password.
