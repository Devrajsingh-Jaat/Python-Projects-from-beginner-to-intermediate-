# 💖 Love Score Calculator

This is a simple Python script that calculates a fun "love score" between two names.

---

## 🧠 What It Does

* Combines two names
* Counts how many letters match:

  * `"true"`
  * `"love"`
* Creates a score by joining the two counts

---

### Step 1: Install Python
