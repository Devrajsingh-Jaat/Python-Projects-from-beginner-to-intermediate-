
# 🔐 Caesar Cipher CLI Tool

A simple Python command-line application to encrypt and decrypt messages using the Caesar Cipher technique.

---

## 📌 Features

- Encode (encrypt) text with a shift value  
- Decode (decrypt) text back to original  
- Keeps non-alphabet characters unchanged  
- Runs in a loop until the user exits  

---

## 🧠 How It Works

The Caesar Cipher shifts each letter by a fixed number.

Example with shift `3`:
- a → d  
- b → e  
- z → c (wraps around)
